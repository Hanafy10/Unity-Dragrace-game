# Unity Drag Race Game - README

## Overview
This is a **Drag Race game built in Unity**. It features fast-paced racing with **multiplayer support**, including both **split-screen multiplayer** and the option to race against **NPC opponents**. The game is designed for Windows and supports keyboard/controller input.

---

## Features
- **Multiplayer Mode:** Play with another player locally using split-screen.
- **Single Player Mode:** Race against AI-controlled NPCs.
- **Drag Racing Mechanics:** Realistic acceleration, gear shifting, and speed physics.
- **Dynamic Tracks:** Multiple drag racing tracks with different lengths and environments.
- **Customizable Cars:** Vehicles with adjustable speed, acceleration, and handling.
- **UI Elements:** Countdown timer, speedometer, lap indicators, and leaderboard.

---

## Installation
1. Make sure you have **Unity Hub** and **Unity version 2021.3 or later** installed.
2. Clone or download this repository:
```bash
git clone https://github.com/Hanafy10/Unity-Dragrace-game.git
```
3. Open Unity Hub and add the project.
4. Open the project in Unity Editor.

---

## How to Play
### Multiplayer (Split-Screen)
1. Select **Multiplayer Mode** from the main menu.
2. Connect the second controller or set up keyboard controls for Player 2.
3. Choose your vehicles.
4. Press **Start** to begin the race. The screen will be split into two views for each player.

### Single Player (NPC Opponents)
1. Select **Single Player Mode**.
2. Choose your vehicle and track.
3. Set the number of NPC racers.
4. Press **Start** to begin the race.

## Setup for Multiplayer & NPC
- **Split-Screen:** Make sure both players are assigned to separate camera views.
- **NPC:** Ensure the AI script is attached to NPC cars, and their speed/difficulty is set in the inspector.

## Dependencies

## How to Build
1. Open the project in Unity Editor.
2. Go to **File > Build Settings**.
3. Select your platform (Windows recommended).
4. Click **Build** and choose a folder to save the executable.
