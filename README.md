Ice Cream Drag Race is a 2D drag-racing university project developed using Unity Engine and C#.
The game features dessert-themed cars, such as an Ice Cream Car and a Banana Car, competing in a 1-mile straight drag race.

The project focuses on speed, timing, acceleration, and race strategy, offering both single-player (Player vs AI) and local two-player split-screen modes.
Bright, colorful, cartoon-style visuals are used to create a fun and engaging racing experience.

This project demonstrates practical skills in :-

- Unity game development

- C# scripting

- Camera systems

- Player & AI controllers

- Game state management

- Winner detection logic
